STARS

STARS is Raytheon Technologies' air traffic control program and has been the main focus for my job for the past 3 years. At this point in time the focus of STARS engineers is to update features and fix bugs within the product. STARS is a massive code base and deals with complex inputs and distributive nodes. As a software engineer on this project, my job is to fix mistakes found both by use of live air traffic controllers and by Ratheon's detailed testing process. We work in an AGILE work environment so there is plenty of team work and cooperation as we work on this complex system.
